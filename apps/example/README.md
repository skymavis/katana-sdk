# Example katana sdk
(Deploy url: tbd)

This is an example katana dex which uses [katana core](https://github.com/skymavis/katana-sdk/tree/main/packages/katana-swap)

## Features

- Show all erc20 tokens available on Katana
- Swap ron/wron
- Swap erc20 tokens

## Requirements

React 18 or later

## Install

```bash
yarn
# or
npm install
```

## Run project

```bash
npm run dev
# or
yarn dev
```
